## Töö kirjeldus
See projekt on HTML5 semantiline “traatkaart” valitud SaaS-teenuse avalehest.  
Lehel on ülemine riba koos brändi ja navigeerimisega, hero-plokk, otsingu ala, funktsiooniplokid, sisublokk koos kõrvalribaga ning jalus.  
Kasutatud on ainult HTML5 semantilisi elemente ja minimaalset CSS-i kastide eristamiseks.  
Sisu, pildid ja tekst ei ole originaalilt kopeeritud.

## Failid repos
- `index.html` – semantiline HTML5 skeem
- `styles.css` – minimaalne CSS kastide piiritlemiseks (kui kasutasid eraldi faili)
- `README.md` – projekti kirjeldus

## Nõuetele vastavus
- `<html lang="et">` olemas  
- Lehel on ainult üks `<h1>`  
- Kasutatud on `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`, `figure`, `figcaption`  
- Skip-link: `<a href="#main">Liigu põhisisu juurde</a>`  
- Leht on loetav ka 320px laiuses vaates  
- Ei kasutatud JavaScripti ega raamistikke  
- CSS on ainult visuaalseks eristamiseks
